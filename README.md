# big_lottery_10_phase
大樂透今年期數開獎號碼統計

## 做這隻程式的原因
家人買大樂透都會想參考大樂透以往中獎的號碼，但用手紀錄不方便，所以做了這程式幫助家人統計號碼

## 需要使用到的module
1. requests
2. bs4
